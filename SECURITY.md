# Security Policy

## Supported versions

Security fixes are applied to the latest release on `main`. Older major versions are unsupported
unless a specific backport is announced.

## Reporting a vulnerability

Please report security issues privately via GitHub Security Advisories for this repository:

https://github.com/deebee-tech/is-helper/security/advisories/new

Include a minimal reproduction when possible. Do not open a public issue for exploitable report
content until a fix is available or maintainers have confirmed disclosure is safe.
